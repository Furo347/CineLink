# Cahier de recettes

Le cahier de recette a jour est disponible dans [docs/RECETTE.md](docs/RECETTE.md).

Ce fichier historique est conserve pour la documentation de certification, mais la source de reference est maintenant le tableau dedie dans `docs/RECETTE.md`, qui couvre :

- inscription et connexion ;
- erreurs d'authentification ;
- chargement prolonge ;
- catalogue, recherche et detail film ;
- favoris ;
- commentaires au bouton et avec le clavier ;
- `Maj + Entree` dans le champ commentaire ;
- follow, feed et profils ;
- role utilisateur standard ;
- role administrateur ;
- suppression de commentaire par admin ;
- suppression utilisateur par admin ;
- refus `401`, `403`, `404` ;
- affichage mobile ;
- rafraichissement direct d'une route React.

Les statuts de recette doivent rester `A verifier` tant que les scenarios n'ont pas ete rejoues manuellement dans l'environnement cible.
